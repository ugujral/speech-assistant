# Speech Assistant

Python app that uses Google's speech recognition and text-to-speech

### Dependencies

```
pip install speechrecognition
pip install pyttsx3
pip install pyaudio
pip install playsound
pip install PyObjC
```

### Voice Commands

You can add other commands, but these are the ones that exist

- What is your name?
- What time is it
- Search
- Find Location
- YouTube
- Stocks
- Exit
